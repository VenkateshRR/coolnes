9. Checking whether the two words are anagrams or not

from collections import Counter
def is_anagram(str1, str2):
return Counter(str1) == Counter(str2)
print(is_anagram('majix', 'magic'))  
print(is_anagram('majix', 'xijam'))
Output:

False
True
